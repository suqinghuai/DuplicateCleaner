# DuplicateCleaner

一个简单实用的重复文件清理工具，专门用于删除带有数字后缀的重复文件（如 "文件(1).txt"、"图片 (2).png" 等）。

## 功能特点

- 自动扫描当前目录中的重复文件
- 识别带有 `(1)`, `(2)`, `(3)` 等数字后缀的文件
- 安全删除重复文件，保留原始文件
- 提供详细的清理报告

## 工作原理

1. 软件会扫描其所在目录中的所有文件
2. 使用正则表达式识别带有数字后缀的文件
3. 删除这些重复文件，保留没有数字后缀的原始文件
4. 显示清理结果，包括删除的文件数量和具体文件列表

## 使用方法

1. 直接双击 `DuplicateCleaner.exe` 文件运行
2. 查看清理结果
3. 按任意键退出程序

## 注意事项

- 脚本会删除带有数字后缀的文件，请确保这些确实是重复文件
- 脚本只会在其所在目录中执行清理操作
- 清理操作不可撤销，请谨慎使用

### 清理前：
```

├── document (1).txt
├── image.png

```

### 清理后：
```
文件夹/

```
